Using Semantic Colors

Use Sass variables in SCSS, CSS custom properties in component CSS, and utility classes in markup.

Role Sass Variables CSS Variables Common CSS Classes
CTA $primary --bs-primary, --bs-primary-rgb .btn-primary, .bg-primary, .text-primary
Active/Interactive $info, $link-color --bs-info, --bs-info-rgb, --bs-link-color-rgb .btn-info, .bg-info, .text-info, .link-info
Structure $secondary, $light, $dark --bs-secondary, --bs-light, --bs-dark .bg-secondary, .bg-light, .text-dark, .border-secondary
