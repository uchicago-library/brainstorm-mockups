Brand Color Families

The following colors are established by the UChicago Brand Guidelines. Except for the Grestone Family which has been tinted into cooler shades of grey for color balance, and have a few shades added.

These family values are Sass tokens by default and can be exposed as CSS variables when runtime usage is needed (see `src\styles\base\_global.scss`). To expose them as utility classes, extend Bootstrap's utility map.

Maroon

Primary brand color reserved for main CTAs and high-priority actions.
