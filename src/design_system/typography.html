Typography & HTML Elements

A comprehensive demonstration of all standard HTML text elements and their styling in the design system.

See A History of the Book for a narrative example of these typography elements in context.

UChicago Sans Serif

This site uses UChicago Sans Serif as the primary typeface, loaded from:

  • Source maintained by UChicago Creative: https://uchicago-brand-fonts.s3.us-east-2.amazonaws.com/UChicagoSansSerif.css
  • Fallback maintained by UChicago Library: https://www.lib.uchicago.edu/web-resources/css/uchicagoSansSerif.css

Font should fallback to system defaults. System fallbacks include: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
