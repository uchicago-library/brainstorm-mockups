HTML Usage

<!-- Include the component script -->
<script src="../components/example-button.js"></script>

<!-- Use the component with different text -->
<example-button>Click Me</example-button>
<example-button>Submit</example-button>
<example-button>Cancel</example-button>

<!-- With attributes -->
<example-button variant="primary">Primary Button</example-button>
<example-button variant="secondary" size="sm">Small Secondary</example-button>
<example-button disabled>Disabled</example-button>
✅ Works without a server! This page can be opened directly with file:// protocol. Web Components are native browser features - no build tools needed.
How it works:
  1. Include example-button.js in your page
  2. Use <example-button>Text</example-button> anywhere
  3. Each instance can have different text and attributes
  4. Update example-button.js once, all instances update automatically
