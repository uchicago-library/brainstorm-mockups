Navbar Component

A responsive, accessible navigation bar built with Bootstrap and Web Components.

Features

  • Responsive Design: Collapses into mobile menu on smaller screens
  • Accessible: Proper ARIA attributes and semantic HTML
  • Customizable: Configurable logo, alt text, and brand text
  • Bootstrap Integration: Uses Bootstrap navbar styles and JavaScript
  • Brand Colors: Automatically uses UChicago brand styling

Component Configuration

üìå Hardcoded Branding

This component has all branding hardcoded for consistency:

  • Logo: UChicago Library white logo (for maroon background)
  • Logo URL: https://www.lib.uchicago.edu/web-resources/img/white-logo.png
  • Alt Text: "University of Chicago Library"
  • Brand Text: "Library"

No attributes needed! Simply include the component and add your navigation links as child elements.

Usage Examples

Basic Navbar with Text

<!-- Include the CSS and component -->
<link rel="stylesheet" href="../components/navbar.css">
<script src="../components/navbar.js"></script>

<!-- Use the component (logo is automatic) -->
<navbar-component>
  <a href="#" class="nav-link active" aria-current="page">Home</a>
  <a href="#" class="nav-link">Collections</a>
  <a href="#" class="nav-link">Services</a>
  <a href="#" class="nav-link">About</a>
</navbar-component>

With Bootstrap Dropdown Menu

<navbar-component>
  <a href="#" class="nav-link">Home</a>
  <div class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" href="#" role="button" 
       data-bs-toggle="dropdown" aria-expanded="false">
      Services
    </a>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">Research Help</a></li>
      <li><a class="dropdown-item" href="#">Document Delivery</a></li>
      <li><hr class="dropdown-divider"></li>
      <li><a class="dropdown-item" href="#">All Services</a></li>
    </ul>
  </div>
  <a href="#" class="nav-link">Contact</a>
</navbar-component>

Accessibility Features

‚úì Built-in Accessibility

  • ARIA labels: Navigation region labeled with aria-label="Main navigation"
  • Keyboard navigation: All links and buttons are keyboard accessible
  • Screen reader support: Toggler button has aria-label and aria-controls
  • Current page indicator: Use aria-current="page" on active link
  • Semantic HTML: Uses <nav> element and proper heading hierarchy
  • Image alt text: Logo always has meaningful alt text
  • Focus indicators: Visible focus states from design system

Responsive Behavior

üì± Mobile-First Design

  • Large screens (‚â•992px): Horizontal navigation with all links visible
  • Medium/Small screens (<992px): Hamburger menu that expands to show links
  • Logo sizing: Automatically adjusts for screen size
  • Touch-friendly: Large tap targets on mobile devices

Customization Tips

  • Add more navigation items by including more <a class="nav-link"> elements
  • Use Bootstrap dropdown classes to create dropdown menus (see examples above)
  • Add a search form or button inside the navbar content slot
  • Mark the current page link with class="nav-link active" aria-current="page"
  • Logo and branding are hardcoded for consistency across the library system
