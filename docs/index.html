University of Chicago Library Design System

A static site for documenting and demonstrating the University of Chicago Library design system, built with Eleventy, Bootstrap 5, and FontAwesome.

Getting Started

Explore the design system through our organized sections:

Component Demos
  • Layouts - Page layout patterns and structures
  • Components - Buttons, forms, cards, accordions, and more
  • Design Tokens - Colors, spacing, borders, shadows
    • Typography - Text styles, headings, lists, tables

Page Mockups
  • Guides in Accordions
  • Guides in Photos
  • Guides in Types

About This Project

This design system documentation site serves multiple purposes:

  • Component Library: Demonstrates reusable UI components (buttons, forms, cards, etc.)
  • Style Guide: Documents the visual design language including colors, typography, and spacing
  • Mockup Showcase: Provides full-page layouts demonstrating how components work together
  • Developer Resource: Offers copy-paste ready HTML/SCSS code for use in other projects

Key Features

Static & Fast

Built with Eleventy for minimal dependencies and maximum performance. Perfect for GitHub Pages hosting.

Bootstrap 5 Foundation

Leverages Bootstrap's powerful SCSS variables and components with custom brand overrides.

Accessible by Design

Follows WCAG 2.1 Level AA standards with semantic HTML, proper ARIA labels, and keyboard navigation.

Highly Reusable

Copy components directly into your projects. All code is modular and well-documented.
