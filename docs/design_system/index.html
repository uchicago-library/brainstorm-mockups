Design System

The University of Chicago Library Design System includes design principles, visual guidelines, components, and patterns.

Overview

This design system is based on the official University of Chicago Library brand guidelines and is designed to be reusable across multiple platforms and services. It balances consistency with flexibility to support diverse use cases.
